Questions fréquemment posées:

1. Comment s'inscrire

Pour vous inscrire, vous devez cliquer sur s'inscrire sur la page d'accueil et remplir les informations requises. Cliquez sur S'inscrire pour créer votre compte.

2. Comment déposer un rapport

Cliquez sur «signaler un problème» sur la page d'accueil et entrez toutes les informations requises pour signaler un problème.

3. Comment s'identifier

Pour vous connecter, vous devez être un utilisateur enregistré. Cliquez sur Connexion depuis la page d'accueil et connectez-vous en utilisant votre nom d'utilisateur et votre mot de passe.

4. Comment nous contacter

Cliquez sur l'onglet «Contactez-nous» de la page d'accueil, qui vous redirigera vers notre page de contact.

5. Et si j'ai oublié mon mot de passe

Si vous avez oublié votre mot de passe, une question de sécurité vous sera posée et un lien vous sera envoyé par e-mail pour réinitialiser votre mot de passe.

6. Pourquoi ne puis-je pas me connecter?

Si vous avez saisi le mot de passe incorrect 3 fois, votre connexion a été interdite pendant une heure pour des raisons de sécurité. Veuillez réessayer dans une heure ou contacter notre service technique.
